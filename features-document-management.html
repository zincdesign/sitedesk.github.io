Features

Document Management

Document Management Features

Flexible ‘single to many’ links – a single document can be linked to one, several, or all your projects, spaces or objects.

Secure platform complies with the latest ISO standards and government data storage requirements.

A single document, attached to multiple elements can be updated in one step.

Digital forms can be exported as PDFs containing embedded images and related documents.

All documents are fully version controlled, creating a complete audit trail.

Document update function can create auto-versioning.

Export full folder structures with documents.

Export fully compliant COBie files.

Fully compliant with PAS-1192.
